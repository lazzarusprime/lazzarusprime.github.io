💾 Save State / 📁 Load State

Save State saves your entire current pattern — all steps, all instruments, and step count — to your browser's local storage. It stays saved even after you close the tab.

Load State restores whatever you last saved. If nothing has been saved yet, it will tell you.

📤 Export JSON / 📥 Import JSON

Export JSON downloads your pattern as a .json file you can back up or share. Use this to save multiple patterns or move them between devices.

Import JSON loads a previously exported .json file back into the grid.
